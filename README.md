# 🛳️ Northwind Traders Dashboard

## 🎯 Problem Statement

The objective of this Microsoft Power BI report is to create a visually appealing and user-friendly dashboard that effectively communicates key performance metrics for Northwind Traders.

The dashboard generates insights into customer behavior, sales patterns, employee performance, inventory trends, and supplier analysis to support data-driven decision-making.

By consolidating data from multiple tables, the dashboard enables stakeholders to explore business data interactively and identify meaningful trends, helping Northwind Traders remain competitive in the wholesale market.

---

# 📂 Dataset Description

The Northwind Database contains sales data for a fictitious company called Northwind Traders, which imports and exports specialty food products worldwide.

---

# 🧾 Tables Used

| Table Name    | Description                                                                                                      |
| ------------- | ---------------------------------------------------------------------------------------------------------------- |
| Customers     | Customer information including company details, contact information, address, city, country, phone, and fax      |
| Employees     | Employee details such as name, designation, hire date, address, reporting manager, and contact information       |
| Orders        | Order-related information including order date, shipped date, freight, shipping details, and customer references |
| Order Details | Product-level transaction details including quantity, unit price, and discount                                   |
| Products      | Product information such as category, supplier, stock, unit price, reorder level, and availability               |
| Suppliers     | Supplier company information including contact details and geographical data                                     |
| Shippers      | Shipping company information                                                                                     |
| Categories    | Product category names and descriptions                                                                          |

---

# 💡 Analytical Questions

* What is the average number of orders per customer?
* Who are the high-value repeat customers?
* How do order patterns vary across countries and cities?
* Which product categories generate the highest revenue?
* What is the relationship between customer location and product category?
* How frequently do different customer segments place orders?
* What is the geographic distribution of employees?
* How do hiring trends vary across employee roles?
* How do product pricing and stock levels impact sales performance?
* Are there any seasonal trends or anomalies in revenue?
* Which suppliers contribute most to product availability and revenue?
